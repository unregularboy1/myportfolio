## 🚀 How to Open Locally

1. Open the `oluwakayode-onasanya-portfolio` folder on your computer
2. Double-click `index.html` — it will open in your default browser
3. Or right-click `index.html` → "Open with" → Chrome

## 🌐 How to Deploy to Netlify

**Drag & Drop (easiest):**
1. Go to https://app.netlify.com/drop
2. Drag the entire `oluwakayode-onasanya-portfolio` folder onto the page
3. Netlify gives you a live URL instantly

**Or zip and upload:**
1. Zip the entire folder
2. Go to https://app.netlify.com → "Add new site" → "Deploy manually"
3. Drop the zip file in

## ✏️ How to Edit Content

- **Name, bio, experience, skills, interests** → Edit `index.html`
- **Colours, fonts, spacing, animations** → Edit `css/style.css`
- **Cursor, scroll effects, nav behaviour** → Edit `js/main.js`
